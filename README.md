# EduStackPro | Web3 E-Learning Static page

**EduStackPro** is a modern, responsive Learning Management System (LMS) designed to bridge the gap between Web2 development and the Decentralized Web (Web3). This version (V1) focuses on a high-end UI/UX, professional search/filter functionality, and Supabase Authentication.

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS (Mobile-First Design)
- **Routing:** React Router Dom (Protected Student Routes)
- **Backend/Auth:** Supabase (PostgreSQL & GoTrue Auth)
- **Icons:** React Bootstrap Icons
- **State Management:** React Hooks (`useState`, `useEffect`, `useLocation`)

## ✨ Key Features
- **Professional Search & Filter:** Instant course filtering by category and title.
- **Dynamic Classroom Layout:** Sticky sidebar and module-based accordion for a focused learning experience.
- **Responsive Branding:** A consistent, high-end "SaaS" aesthetic across all device sizes.
- **Secure Authentication:** User registration and login powered by Supabase.

## 📈 Roadmap for V2 (Web3 Integration)
- **On-Chain Enrollment:** Purchasing courses using ETH/MATIC via Wagmi/Ethers.js.
- **NFT Certifications:** Verifiable completion certificates minted on the Polygon network.
- **Smart Contract Integration:** Connecting the frontend to Solidity-based course logic.
